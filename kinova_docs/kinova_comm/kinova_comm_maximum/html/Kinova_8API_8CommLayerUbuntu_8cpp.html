int  ExpectedMessageIn,
int &  ReceivedQtyIn 
)

This function read data on the device's internal RS-485 BUS via USB.

Parameters
PackagesInA list of RS485_Message where the data will be copied(OUTPUT).
ExpectedMessageInHow many message you are expecting to receive(INPUT).
ReceivedQtyInHow many you've really received(OUTPUT).
Returns
  • NO_ERROR_KINOVA if operation is a success
  • ERROR_OPERATION_INCOMPLETED Means that everything went fine but no data was received.
  • ERROR_RS485_IO_PENDING Means that there was a data overflow during the communication.
  • ERROR_RS485_TIMEOUT Means that the application times out after COMMUNICATION_TIME_OUT ms.
  • UNKNOWN_ERROR Means that the communication went wrong but we don't know why.
