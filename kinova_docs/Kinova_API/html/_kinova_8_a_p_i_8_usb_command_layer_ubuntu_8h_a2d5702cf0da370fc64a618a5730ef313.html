KINOVAAPIUSBCOMMANDLAYER_API int SetTorqueVelocityLimitFilter ( float  Command[COMMAND_SIZE])

This function sets the filter for the estimation of the velocity used for the maximum velocity safety feature. The parameters will reset to default values every time the robotic arm is rebooted.

Parameters
CommandThe feed to each actuator
Warning
These functions should only be used by advanced users. The user should be very cautious and be ready to turn off the robot if a problem occurs. The user should quit torque control mode and be in Trajectory-Position control mode before using theses functions. Do not use these functions if you don't know what you are doing.
